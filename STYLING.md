# ERP Finance Frontend — Enhanced UI

Beautiful, modern Angular frontend with creative styling inspired by the prototype HTML but taken to the next level with:
- Gradient backgrounds and glassmorphism effects
- Smooth animations and transitions
- Modern card-based layouts
- Color-coded data visualization
- Responsive grid systems
- Interactive hover states

## Run
```powershell
cd "c:\D_Drive\INTERNSHIP\Fintech.Task 2\Erp_Finance\frontend"
npm install
npm start
```

Visit http://localhost:4200

Ensure backend is running at http://localhost:8080 for API calls.

## Features
- Dashboard with stat cards and quick actions
- Accounts management with color-coded types
- Journal entry creation with balance validation
- Trial balance with accounting equation check
- Bank reconciliation with match scoring

## Styling Highlights
- CSS variables for consistent theming
- Gradient navigation with backdrop effects
- Animated cards with hover transformations
- Form inputs with focus effects
- Color-coded financial data
- Responsive breakpoints for mobile
