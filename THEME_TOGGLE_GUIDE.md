# 🌓 Theme Toggle - Quick Start Guide

## 🎯 Where to Find It

The theme toggle button is located in the **top-right corner** of the application:
```
Topbar → User Actions Section → Between Notifications and User Profile
```

Visual location:
```
[ERP Finance Module]     [🔔 3] [🌓 Toggle] [👤 User Profile]
```

---

---

## 🚀 How to Use

### **Step 1: Find the Toggle**
Look at the top-right area of your screen, next to the bell icon (🔔)

### **Step 2: Click the Toggle**
- Click anywhere on the toggle button
- See the slider smoothly animate to the other side
- Watch as the entire app transforms instantly

### **Step 3: Enjoy Your Theme**
- Your preference is automatically saved
- Next time you visit, your theme choice is remembered
- Toggle as many times as you want!

## 🆘 Quick Help

### **Can't see the toggle?**
- Look in the top-right corner, near the bell icon
- It's between notifications and your user profile

### **Toggle not working?**
- Try refreshing the page
- Check if JavaScript is enabled
- Clear browser cache if needed

### **Theme not saving?**
- Browser must allow localStorage
- Don't use private/incognito mode
- Check browser settings
